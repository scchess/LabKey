Luke is a Lucene index browser. This version is 4.7.0 of a forked version of the official Luke (which hasn't been updated to recent Lucene
versions). https://github.com/DmitryKey/luke

See luke.bat for a sample command line invocation.

Browse to select your Lucene index directory, for example:

  c:\tomcat\temp\labkey_full_text_index
