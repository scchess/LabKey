One aim of this project is to generate high quality catalogs of reference variants for rhesus macaques. The variant catalogs below are generated from joint genotyping of our cohort of high-depth whole-genome sequencing data, using a modified version of the GATK Best Practice Variant Discovery Pipeline. Each release is available either as a direct download (comparable to the dbSNP human resource bundles), or these data can be explored through our genome browser using the links below.
